Enums

Unlike Java enums, C# enums cannot contain fields/properties or methods. However, they can have extensions,

Unlike Java, enums have a backing value type (default int) and can be cast (eg (SomeEnum) 10 and (int) SomeEnumMember). These casts should usually be avoided

The backing type can be changed with : syntax

Each enum member can also be assigned a specific value:

enum Color : ushort {
    Red,
    Green = 10,
    Blue
}

In this case, Red is 0, Green is 10, and Blue is 11
